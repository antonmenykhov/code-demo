## Как обновить набор в существующем проекте

1. Скачать набор Набор компонентов отдельно
2. Перейти в папку node_modules проекта
3. Удалить содержимое папки .vite
4. Очистить папку tnnc-ui-kit
5. Распакавать содежримое папки package скаченного архива в папку tnnc-ui-kit
