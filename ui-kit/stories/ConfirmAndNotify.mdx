# Confirm

Функция диалог, принимает вопрос, показывает пользователю, возвращает промис. \
Если пользователь нажимает ДА промис резолвится\
Если пользователь нажимает НЕТ промис режектится

<code>
  confirm('Закрыть диалог?').then(()=>\{console.log('Пользователь нажал ДА')\}
  ).catch(()=>\{console.log('Пользователь нажал нет')\})
</code>

# Notify

Функция, генерирует уведомление пользователю, принимает текст уведомления, тип, продолжительность анимации показа.

Тип и продолжительность необязательны, по умолчанию 'success' и 3000

<code>notify('Сообщение пользователию', 'success', 3000)</code>
<code>notify('Сообщение пользователию', 'danger', 3000)</code>
